Formatação

Texto Marcado

Para adicionar uma marcação em um texto utiliza-se a tag MARK.

Texto deletado

Para adicionar um texto deletado que deve ser ignorado pelo leitor,
utiliza-se a tag DEl.

Texto Destacado

O texto destacado serve para mostrar que tal palavra é uma coisa importante,
e para isso utiliza-se a tag STRONG.

Texto Itálico ou Ênfase

Para adicionar uma formatação itálica utiliza-se a tag EM.

Texto Inserido

O Texto inserido é o contrario do texto deletado, ele não deve ser ignorado pelo leitor,
e para isso utiliza-se a tag INS.

Texto Sobrescrito

Texto Sobrescrito é para usar em contas matemáticas por exemplo, X20 +3 e a tag que é utilizada SUP.
Ex: "Essa frase fica acima das frases normais."

Texto Subscrito

Texto subscrito é para usar em elementos químicos por exemplo, H20
a tag para isso é SUB, a tag SUB também pode ser usada para deixar
as letras pequenas, fazer uma frase pequena.
Ex: "Depois de clicar no botão não tem volta."

Hierarquia dos Títulos

Subtitulo H2

Subtitulo do Subtitulo, H3

Subtitulo dos outros Subtitulos, H4

Subtitulo do Subtitulo H4, H5
E por fim o H6 o menor na Hierarquia dos titulos, H6
